# TECASM

TEC-8 在线汇编

---

## 语法说明

### 注释

以 `;` 开头，直到行末。

```
; 这是注释
NOP  ; 行尾注释
```

### 标签

标签以字母或下划线开头，以 `:` 结尾，不区分大小写。标签可以单独占一行，也可以放在指令前。

```
START:
LOOP: INC R0
```

### 数字格式

| 格式 | 示例 | 说明 |
|------|------|------|
| 十进制 | `42` | 无后缀 |
| 十六进制 | `2AH` | `H` 后缀，不区分大小写 |
| 二进制 | `00101010B` | `B` 后缀，只含 0/1 |

### 寄存器

共四个通用寄存器：`R0` `R1` `R2` `R3`，不区分大小写。

## 指令集

### 运算指令

| 指令 | 格式 | 说明 |
|------|------|------|
| `NOP` | `NOP` | 空操作 |
| `ADD` | `ADD Rd, Rs` | Rd ← Rd + Rs |
| `SUB` | `SUB Rd, Rs` | Rd ← Rd − Rs |
| `AND` | `AND Rd, Rs` | Rd ← Rd & Rs |
| `INC` | `INC Rd` | Rd ← Rd + 1 |

### 访存指令

| 指令 | 格式 | 说明 |
|------|------|------|
| `LD` | `LD Rd, [Rs]` | Rd ← mem[Rs] |
| `ST` | `ST Rs, [Rd]` | mem[Rd] ← Rs |

### 跳转指令

跳转目标可以是标签或立即数。`JC`/`JZ` 使用相对寻址，偏移量为 4 位有符号数（范围 −8 ~ 7）。

| 指令 | 格式 | 说明 |
|------|------|------|
| `JC` | `JC 标签/地址` | 进位标志为 1 时跳转 |
| `JZ` | `JZ 标签/地址` | 零标志为 1 时跳转 |
| `JMP` | `JMP [Rd]` | 间接跳转，PC ← mem[Rd] |

### 其他指令

| 指令 | 格式 | 说明 |
|------|------|------|
| `OUT` | `OUT Rs` | 输出寄存器值 |
| `DI` | `DI` | 禁用中断 |
| `EI` | `EI` | 启用中断 |
| `IRET` | `IRET` | 中断返回 |
| `STOP` | `STOP` | 停机 |

### 伪指令

| 伪指令 | 格式 | 说明 |
|--------|------|------|
| `DATA` | `DATA 数字` | 在当前地址放置一个字节 |

`DATA` 接受十进制（含负数）、十六进制（`H` 后缀）和二进制（`B` 后缀）格式。十进制范围 −128 ~ 255，负数以补码存储。

```
TABLE:
  DATA 42
  DATA 2AH
  DATA 00101010B
  DATA -1    ; FFH
```
